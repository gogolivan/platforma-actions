= Platforma Actions


== Getting Started 
Clone official GitHub Actions TypeScript template
[source, text]
----
git clone https://github.com/actions/typescript-action.git platforma-pull-request-action --depth 1 && rm -rf ./platforma-pull-request-action/.git
----

1. Navigate to the cloned directory.
2. Create `.nvmrc` file and set the Node.js version from `.node-version` file.
3. Configure `package.json` file to set name, description, author, repository, keywords, etc.
4. Read `README.md` file for more details on how to build and test the action.
